# How to Extend the Foundation

1. Read `llmdoc/reference/repo-layout.md`.
2. Confirm whether the change belongs in `apps/desktop` or one of the Rust crates.
3. Update `docs/architecture.md` if the ownership boundary changes.
4. Update `docs/roadmap.md` if phase scope changes.
5. Add or revise `llmdoc/reference/` docs for stable contracts.
6. Record foundational scope changes under `llmdoc/memory/decisions/`.
